## Mandatory Subsections

### 报告元信息

Include at least:

- Analysis date
- Current price and date
- Market cap
- Holding channel
- Dividend-tax assumption
- Risk-free rate
- Source files used
- Accounting standard
- Report currency and unit assumptions

### 一、Executive Summary（执行摘要）

Summarize:

- Final recommendation
- Core thesis
- Whether veto gates were triggered
- Key return and hurdle figures
- Position-sizing conclusion

### 1.5 财务趋势速览（近5年）

Provide a compact multi-year trend view before factor analysis.
